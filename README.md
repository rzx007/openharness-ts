# OpenHarness-ts

OpenHarness жҳҜдёҖдёӘејҖжәҗ AI Agent жЎҶжһ¶пјҢжҸҗдҫӣзұ» Claude Code зҡ„дәӨдә’ејҸзј–з ҒеҠ©жүӢдҪ“йӘҢгҖӮжң¬йЎ№зӣ®жҳҜе…¶ TypeScript еӨҚеҲ»е®һзҺ°пјҢж ёеҝғ harnessпјҲеј•ж“Һ / е·Ҙе…· / жқғйҷҗ / дјҡиҜқ / TUIпјүе·ІеҸҜз”ЁпјҢд»ҚеңЁжҢҒз»ӯеҗ‘ Python еҺҹзүҲ v0.1.9 еҜ№йҪҗдёӯгҖӮ

## зү№жҖ§

> вҡ пёҸ жң¬йЎ№зӣ®д»ҚеңЁеӨҚеҲ»дёӯгҖӮдёӢиЎЁж ҮжіЁеҗ„иғҪеҠӣзӣёеҜ№ Python еҺҹзүҲ **v0.1.9** зҡ„**зңҹе®һзҠ¶жҖҒ**пјҡвң… еҹәжң¬еҜ№йҪҗ В· рҹҹЎ еҸҜз”ЁдҪҶз®ҖеҢ– В· рҹҹ  йӘЁжһ¶/йғЁеҲҶ В· рҹ”ҙ жңӘе®һзҺ°гҖӮе®Ңж•ҙе·®и·қжё…еҚ•дёҺиЎҘйҪҗи·Ҝзәҝи§Ғ [PLAN-REMAINING.md](PLAN-REMAINING.md)гҖӮ

- вң… **еӨҡжЁЎеһӢж”ҜжҢҒ** вҖ” 20 дёӘ Provider иҮӘеҠЁжЈҖжөӢпјҲAnthropic еҺҹз”ҹ + OpenAI е…је®№пјүпјҢеҗ« `<think>` еқ—иҝҮж»ӨгҖҒеӣҫзүҮ/vision дј йҖ’гҖҒgpt-5/o зі»еҲ— token еӯ—ж®өйҖӮй…ҚгҖӮрҹҹЎ жҡӮзјә Codex/Copilot и®ўйҳ…гҖҒreasoning effort
- рҹҹЎ **еҶ…зҪ®е·Ҙе…·пјҲ41пјү** вҖ” ж–Үд»¶ / Bash / Web / Grep / Cron / MCP / Task / Agent / TaskWait зӯүйҪҗе…ЁпјҢbash/grep/glob еҒҘеЈ®жҖ§е·ІеҜ№йҪҗ v0.1.8пјҲи¶…ж—¶дҝқз•ҷиҫ“еҮәгҖҒиҝӣзЁӢз»„жқҖйҷӨгҖҒgitignore/и¶…й•ҝиЎҢеӨ„зҗҶпјүпјӣжҡӮж— еӣҫзүҮзұ»е·Ҙе…·
- рҹҹЎ **еӨҡ Agent зј–жҺ’** вҖ” `agent` е·Ҙе…·еҸҜзңҹе®һжҙҫеҸ‘еӯҗиҝӣзЁӢ teammateпјҡзӢ¬з«Ӣ git worktree йҡ”зҰ»гҖҒеҸӘиҜ»е·Ҙе…·иҮӘеҠЁж”ҫиЎҢгҖҒ`TaskWait` йҳ»еЎһеҸ–з»“жһңгҖҒTUI жҳҫзӨә teammate зҠ¶жҖҒпјӣCoordinator 7 дёӘ agent + XML д»»еҠЎйҖҡзҹҘе°ұз»ӘгҖӮжҡӮзјәеҶҷж“ҚдҪңиҪ¬ leader е®Ўжү№гҖҒеӨҡиҪ® workerгҖҒsequential/parallel/pipeline и°ғеәҰ
- вң… **MCP еҚҸи®®** вҖ” stdio + HTTP(streamable)/SSE дј иҫ“иҝһжҺҘеӨ–йғЁ MCP ServerпјҢж”ҜжҢҒ headers йүҙжқғгҖҒеӨұиҙҘйҡ”зҰ»пјӣMCP OAuth жөҒзЁӢеҫ…иЎҘ
- вң… **жқғйҷҗзі»з»ҹ** вҖ” default / plan / full_auto + е·Ҙе…·й»‘зҷҪеҗҚеҚ•гҖҒи·Ҝеҫ„и§„еҲҷгҖҒе‘Ҫд»ӨжӢ’з»қпјӣswarm worker еҸӘиҜ»е·Ҙе…·иҮӘеҠЁж”ҫиЎҢпјӣTUI дёӢ Edit/Write ж”№ж–Үд»¶еүҚжҳҫзӨә unified diff йў„и§ҲпјҢеҸҜжң¬ж¬Ў/ж•ҙдёӘдјҡиҜқжү№еҮҶ
- вң… **Hook з”ҹе‘Ҫе‘Ёжңҹ** вҖ” 10 зұ»дәӢд»¶гҖҒpriority жҺ’еәҸгҖҒcommand/http/prompt/agent еӣӣз§Қзұ»еһӢгҖҒmatcher иҝҮж»ӨгҖҒ`$ARGUMENTS` жіЁе…Ҙ+shell иҪ¬д№ү
- рҹҹЎ **дјҡиҜқжҢҒд№…еҢ–** вҖ” Session еӯҳеӮЁ / `--continue` / `--resume` / CronпјҲеқҮдёәеҹәзЎҖзүҲпјү
- рҹҹ  **жҸ’д»¶зі»з»ҹ** вҖ” еҸҜиҜ» `plugin.json`пјӣе·Ҙе…·иҮӘеҠЁеҸ‘зҺ°дёҺ commands/agents/hooks иҙЎзҢ®еҠ иҪҪеҫ…иЎҘ
- рҹҹ  **Channel йҖӮй…ҚеҷЁ** вҖ” Stdio / HTTP / йЈһд№ҰпјҲеҹәзЎҖпјүпјӣTelegram/Discord/Slack зӯүеӨҡйҖҡйҒ“гҖҒйҷ„д»¶гҖҒзҫӨз»„и·Ҝз”ұеҫ…иЎҘ
- рҹҹЎ **TUI еүҚз«Ҝ** вҖ” React/Ink з»Ҳз«Ҝ UIпјҢеҠ©жүӢж¶ҲжҒҜ Markdown жёІжҹ“пјҲж Үйўҳ/еҲ—иЎЁ/д»Јз Ғеқ—й«ҳдә®/иЎЁж јпјүгҖҒSwarmPanel жҳҫзӨәеӯҗ agent зҠ¶жҖҒгҖҒеҗҜеҠЁжё…еұҸгҖҒEdit/Write жқғйҷҗжЎҶ unified diff йў„и§ҲпјҲ`[y]`жң¬ж¬Ў/`[a]`ж•ҙдёӘдјҡиҜқ/`[n]`жӢ’з»қпјүпјӣиҜӯжі•й«ҳдә®гҖҒoutput style еҫ…иЎҘ
- рҹ”ҙ **е°ҡжңӘеӨҚеҲ»** вҖ” `personalization`пјҲзҺҜеўғдәӢе®һжҠҪеҸ–пјүгҖҒ`ohmo`пјҲдёӘдәәеҠ©зҗҶ + еӨҡжё йҒ“зҪ‘е…іпјүгҖҒ`sandbox`пјҲDocker йҡ”зҰ»пјҢеҪ“еүҚдёәеҚ дҪҚпјү
- вӣ” **дёҚеңЁеӨҚеҲ»иҢғеӣҙ** вҖ” `autopilot`пјҲд»“еә“зә§иҮӘеҠЁй©ҫй©¶ + dashboardпјү

## еҝ«йҖҹејҖе§Ӣ

### зҺҜеўғиҰҒжұӮ

- Node.js >= 18
- pnpm >= 10
- Bun >= 1.0пјҲCLI йҖҡиҝҮ Bun жһ„е»ә/иҝҗиЎҢпјү

### е®үиЈ…

```bash
git clone <repo-url> OpenHarness-ts
cd OpenHarness-ts
pnpm install
```

### жһ„е»ә

```bash
pnpm build
```

### жөӢиҜ•

```bash
pnpm test
```

### иҝҗиЎҢВ·

```bash
# и®ҫзҪ® API KeyпјҲжҢүжүҖз”Ё Provider йҖүжӢ©пјҢи§ҒдёӢж–№вҖңй…ҚзҪ®вҖқпјү
export ANTHROPIC_API_KEY="sk-ant-..."

# CLI е®үиЈ…еҗҺжҸҗдҫӣдёӨдёӘзӯүд»·е‘Ҫд»Өпјҡohs дёҺ openharness
```

### ејҖеҸ‘йҳ¶ж®өиҝҗиЎҢ

```bash

# еҸ–ж¶Ҳй“ҫжҺҘпјҡpnpm unlink --global

# ж–№ејҸдәҢпјҡBun зӣҙи·‘жәҗз ҒпјҲж”№д»Јз Ғз«ӢеҲ»з”ҹж•ҲпјҢж— йңҖ buildпјү
```

### и®ҫзҪ® API Key

**ж–№ејҸдёҖпјҡCLIпјҲжҺЁиҚҗпјүвҖ”вҖ” еӯҳиҝӣ `~/.openharness/credentials.json`пјҢж— йңҖжүӢж”№ж–Үд»¶**

```bash
ohs setup                                   # дәӨдә’еҗ‘еҜјпјҡйҖү provider вҶ’ иҫ“ key вҶ’ йҖү model
# жҲ–йқһдәӨдә’зӣҙжҺҘй…Қпјҡ
ohs provider add deepseek -k sk-xxxx --use --model deepseek-v4-flash
ohs provider list                           # жҹҘзңӢ provider + key жқҘжәҗпјҢж ҮжіЁ active
ohs doctor                                  # йӘҢиҜҒ key жқҘжәҗ
ohs --dry-run                               # йў„и§Ҳи§ЈжһҗеҗҺзҡ„иҝҗиЎҢж—¶й…ҚзҪ®(дёҚи°ғжЁЎеһӢ)
```

**ж–№ејҸдәҢпјҡзҺҜеўғеҸҳйҮҸ**

**Linux / macOSпјҡ**

```bash
# еҪ“еүҚдјҡиҜқ
export ANTHROPIC_API_KEY="sk-ant-..."

# жҢҒд№…еҢ–пјҲеҶҷе…Ҙ shell й…ҚзҪ®ж–Үд»¶пјү
echo 'export ANTHROPIC_API_KEY="sk-ant-..."' >> ~/.bashrc
source ~/.bashrc
```

**Windows PowerShellпјҡ**

```powershell
# еҪ“еүҚдјҡиҜқ
$env:ANTHROPIC_API_KEY = "sk-ant-..."

# жҢҒд№…еҢ–пјҲеҶҷе…Ҙз”ЁжҲ·зҺҜеўғеҸҳйҮҸпјү
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-...", "User")

# жҢҒд№…еҢ–еҗҺйҮҚеҗҜз»Ҳз«Ҝз”ҹж•ҲпјҢжҲ–з«ӢеҚіз”ҹж•Ҳпјҡ
$env:ANTHROPIC_API_KEY = [Environment]::GetEnvironmentVariable("ANTHROPIC_API_KEY", "User")
```

**Windows CMDпјҡ**

```cmd
:: еҪ“еүҚдјҡиҜқ
set ANTHROPIC_API_KEY=sk-ant-...

:: жҢҒд№…еҢ–пјҲзі»з»ҹзҺҜеўғеҸҳйҮҸпјү
setx ANTHROPIC_API_KEY "sk-ant-..."
```

д№ҹеҸҜд»ҘйҖҡиҝҮ `settings.json` жҲ– `--api-key` еҸӮж•°дј е…ҘпјҢдјҳе…Ҳзә§пјҡCLI еҸӮж•° > зҺҜеўғеҸҳйҮҸ > й…ҚзҪ®ж–Үд»¶ > й»ҳи®ӨеҖјгҖӮ

### зҺҜеўғеҸҳйҮҸ


| еҸҳйҮҸ                       | иҜҙжҳҺ                           |
| ------------------------ | ---------------------------- |
| `ANTHROPIC_API_KEY`      | Anthropic API Key            |
| `OPENAI_API_KEY`         | OpenAI API Key               |
| `DEEPSEEK_API_KEY`       | DeepSeek API Key             |
| `ZHIPU_API_KEY`          | жҷәи°ұ AIпјҲGLMпјүAPI Key            |
| `OPENHARNESS_CONFIG_DIR` | иҮӘе®ҡд№үй…ҚзҪ®зӣ®еҪ•пјҲй»ҳи®Ө `~/.openharness`пјү |
| `OPENHARNESS_MODEL`      | й»ҳи®ӨжЁЎеһӢеҗҚз§°                       |
| `OPENHARNESS_BASE_URL`   | йҖҡз”Ё API Base URL иҰҶзӣ–пјҲ**жүҖжңү provider**пјү  |
| `OPENHARNESS_API_FORMAT` | API ж јејҸпјҲanthropic / openaiпјү   |

> вҡ пёҸ `ANTHROPIC_BASE_URL` д»… Anthropic provider з”ҹж•ҲпјҲз”ұ Anthropic SDK иҮӘиЎҢиҜ»еҸ–пјүпјҢ**дёҚдјҡ**еҪұе“Қ deepseek/openrouter зӯүе…¶е®ғ providerвҖ”вҖ”иҰҒе…ЁеұҖиҰҶзӣ– baseURL иҜ·з”Ё `OPENHARNESS_BASE_URL`гҖӮ

## Provider й…ҚзҪ®зӨәдҫӢ

### DeepSeek

DeepSeek дҪҝз”Ё OpenAI е…је®№ж јејҸпјҢжЎҶжһ¶дјҡж №жҚ® `provider: deepseek` жҲ– `deepseek` жЁЎеһӢ/еҹҹеҗҚе…ій”®еӯ—иҮӘеҠЁжЈҖжөӢгҖӮеҪ“еүҚжЁЎеһӢпјҡ`deepseek-v4-flash`гҖҒ`deepseek-v4-pro`пјҲж—§еҗҚ `deepseek-chat`/`deepseek-reasoner` е°ҶдәҺ 2026-07 ејғз”ЁпјүгҖӮ

**ж–№ејҸдёҖпјҡCLIпјҲжҺЁиҚҗпјү**

```bash
ohs provider add deepseek -k sk-xxxxxxxx --use --model deepseek-v4-flash
```

**ж–№ејҸдәҢпјҡзҺҜеўғеҸҳйҮҸ**

```bash
export DEEPSEEK_API_KEY="sk-xxxxxxxxxxxxxxxx"
```

**ж–№ејҸдәҢпјҡsettings.json**

```json
{
  "model": "deepseek-chat",
  "apiFormat": "openai",
  "baseUrl": "https://api.deepseek.com",
  "apiKey": "sk-xxxxxxxxxxxxxxxx"
}
```

**ж–№ејҸдёүпјҡCLI еҸӮж•°**

```bash
ohs --model deepseek-chat \
```

**еҸҜз”ЁжЁЎеһӢпјҡ** `deepseek-chat`пјҲDeepSeek-V3пјүгҖҒ`deepseek-reasoner`пјҲDeepSeek-R1пјҢж”ҜжҢҒ reasoning_contentпјү

### иҮӘеҠЁжЈҖжөӢи§„еҲҷ

жЎҶжһ¶ж”ҜжҢҒдёүзә§иҮӘеҠЁжЈҖжөӢпјҢж— йңҖжүӢеҠЁжҢҮе®ҡ providerпјҡ


| жЈҖжөӢзә§еҲ«         | и§„еҲҷ              | зӨәдҫӢ                                          |
| ------------ | --------------- | ------------------------------------------- |
| API Key еүҚзјҖ   | еҢ№й…Қ `sk-` еҗҺзҡ„зү№еҫҒеӯ—з¬Ұ | Anthropic: `sk-ant-`                        |
| Base URL е…ій”®еӯ— | еҢ№й…ҚеҹҹеҗҚе…ій”®иҜҚ         | DeepSeek: `deepseek.com`пјҢGLM: `bigmodel.cn` |
| жЁЎеһӢеҗҚз§°е…ій”®еӯ—      | еҢ№й…ҚжЁЎеһӢеҗҚеүҚзјҖ/е…ій”®иҜҚ     | DeepSeek: `deepseek-`*пјҢGLM: `glm-`*         |


еӣ жӯӨеңЁи®ҫзҪ®еҘҪеҜ№еә”зҺҜеўғеҸҳйҮҸеҗҺпјҢйҖҡеёёеҸӘйңҖжҢҮе®ҡ `--model` еҚіеҸҜпјҡ

```bash
# DeepSeek вҖ” иҮӘеҠЁжЈҖжөӢпјҲDEEPSEEK_API_KEY е·Іи®ҫзҪ®пјү
ohs --model deepseek-chat "hello"

# GLM вҖ” иҮӘеҠЁжЈҖжөӢпјҲZHIPU_API_KEY е·Іи®ҫзҪ®пјү
ohs --model glm-4-plus "hello"
```
